Riders republic 2 Mobile

For Android and iOS

Size

3.5GB (Installer) 5GB (Post-Download Cache)

Price

Free

Android Compatible

Qualcomm Snapdragon 845 and better, 4GB RAM

iOS Devices Required

iPhone XS Max or newer, iPad Pro 2018 or later

Installation Guide

Android
iOS
  • Download the APK file using the Android download button above
  • Go to Settings > Security > Enable "Unknown Sources"
  • Locate the downloaded APK file in your file manager
  • Tap to install and follow on-screen instructions
  • After installation, download the additional game cache files
  • Place cache files in Android/obb/ folder
  • Launch the game and enjoy!
  • Download the IPA file using the iOS download button
  • Install AltStore on your computer (Windows/Mac required)
  • Connect your iOS device and open AltStore
  • Select "Install .ipa" and choose the downloaded file
  • Enter your Apple ID when prompted
  • Trust the developer in Settings > General > Device Management
  • Launch the game and download additional resources

Features

  • Extreme Sports Physics: Realistic physics for biking, snowboarding, skiing, and wingsuit flying
  • Massive Open Worlds: Explore mountains, valleys, and urban playgrounds for endless action
  • Dynamic Weather: Experience changing conditions like snow, rain, and sunny days that affect gameplay
  • Multiplayer Mode: Compete with friends or players worldwide in real-time races and challenges
  • Customization: Personalize your rider with outfits, gear, and equipment
